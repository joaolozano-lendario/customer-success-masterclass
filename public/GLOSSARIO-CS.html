Formulas Rapidas

Metrica Formula
Churn Rate (Clientes Perdidos / Clientes Inicio) x 100
Retention Rate 100% - Churn Rate
LTV Receita Media Mensal / Churn Rate
CAC (Marketing + Vendas) / Clientes Adquiridos
NPS % Promotores - % Detratores
MRR Soma de todas as mensalidades
ARR MRR x 12
Expansion MRR Upsells + Cross-sells - Downgrades

Benchmarks Importantes

Churn Rate B2B SaaS

<3% = Excelente

3-7% = Normal

NPS

>+50 = Excelente

0 a +50 = Bom

LTV/CAC

3:1 = Bom

5:1 = Excelente

Time to Value

<7 dias = Ideal

<30 dias = Bom

Renewal Rate

>95% = Excelente

>90% = Bom

NRR (Net Revenue Retention)

>100% = Excelente

90-100% = Bom
