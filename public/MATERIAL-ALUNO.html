Metrica: Churn Rate

1. CHURN RATE

Taxa de cancelamento

Churn = (Clientes Perdidos / Clientes Inicio) x 100
<3% = Excelente 3-7% = Normal >10% = Alerta

2. NPS (Net Promoter Score)

"De 0 a 10, quanto voce recomendaria?"

9-10 Promotores 7-8 Neutros 0-6 Detratores

NPS = % Promotores - % Detratores

3. HEALTH SCORE

Pontuacao de saude do cliente

80-100 = Saudavel
50-79 = Atencao
0-49 = Risco

4. EXPANSION MRR

Expansion = Upsells - Downgrades

Meta: Sempre positivo

5. TIME TO VALUE

Tempo ate primeira vitoria

<7 dias = Ideal
